Important: This tool makes direct API calls to Anthropic from your browser, which exposes your API key to the webpage you're visiting. Only use this on websites you trust completely.

CORS Limitations: Many websites (including major news sites, social media, and corporate sites) will block the API calls due to CORS policies. This is a security feature, not a bug.

Sites that typically work: Personal blogs, documentation sites, GitHub pages, and other sites without strict CORS policies.

How to use:

  1. Drag the bookmarklet below to your bookmarks bar
  2. Navigate to any webpage (preferably one you trust)
  3. Click the bookmarklet to launch the agent
  4. Enter your Anthropic API key when prompted
  5. Start chatting with the agent about the webpage

Privacy

Your API key is stored locally in your browser and never sent to our servers. All communication happens directly between your browser and Anthropic's API.
